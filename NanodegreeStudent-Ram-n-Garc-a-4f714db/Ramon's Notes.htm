Box Positioning

HTML elements are boxes and each box has 4 components. Because there are so many components to each box, two techniques exist to help deal with the sizing issues that could arise:
1) Set sizes in terms of percent rather than pixels.
2) Set the box-sizing attribute to border-box for every element.
We also have to take into consideration that Divs are block elements (as opposed to inline ), so by default they take up the entire width of a page. Adding the rule "display: flex; to the appropiate CSS will override this behavior and let divs appear next to each other. You can find more guidance on boxes at a guide to flexbox
